```

  → Analytics & Reporting
```

### Key Database Objects

**Dimension Tables**:
- **DIM_PRODUCT** - Product master data (11 products)
  - Includes: ASIN, parent_asin, parent_name, SKU, brand, product_type
  - Cost data: COGS, shipping_cost, FBA_cost ✅ (all populated including LolliME)

**Performance Tables**:
- **SQP_ASIN_View_Simple_Week** - Search Query Performance (weekly aggregated)
  - 44,568 rows across 69 weeks
  - Metrics: Impressions, clicks, cart adds, purchases by search query
  - Primary product focus: B09XQ56RK5 (Purple LolliBox)

- **SCP_ASIN_View_Week** - Search Catalog Performance (weekly aggregated)
  - ASIN-level performance metrics
  - Search funnel data (impressions → clicks → purchases)

**Interface Views**:
- **V_SRC_Products** - Unified product data from Fivetran
- **V_SRC_AmazonAds_campaign_history** - Campaign performance with temporal versioning
- **V_SRC_AmazonAds_keyword** - Keyword targeting data
- **V_SRC_AmazonAds_SearchTerms** - Search term performance with campaign context
- **V_SRC_AmazonAds_purchased_product** - Product purchase attribution
- **V_UNIFIED_SCP_DATA** - Unified Search Catalog Performance data
- **V_UNIFIED_INVENTORY_SNAPSHOT** - FBA inventory data

**Stored Procedures**:
- **SP_MERGE_PRODUCT_DIM** - Merges products from V_SRC_Products
