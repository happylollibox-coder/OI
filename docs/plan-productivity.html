DevOps & Deployment

3 tasks
GitHub Actions CI/CD Larger ~3 hrs click to expand
Auto-lint on PRs, auto-deploy on merge. No more manual deploys.
eliminates deploy errors

Three workflows:

  • On PR: Lint (ESLint, ruff, sqlfluff), type-check, run tests
  • On merge to main: Deploy BigQuery views, build + deploy Cloud Run, deploy Cloud Functions
  • Scheduled: Data freshness health check via config.yaml monitoring settings
Unified Deploy Command Larger ~2 hrs click to expand
One command to deploy everything: make deploy-all
replaces 6+ scripts
  • Deploy BigQuery objects, Cloud Run app, Cloud Functions in one go
  • Validate with deployment/validate.sh
  • Flags: --dry-run, --bq-only, --app-only, --functions-only
BigQuery Migration Runner Larger ~2 hrs click to expand
Versioned, ordered migrations with history tracking. No more guessing what ran.
prevents migration chaos
  • Version-prefixed migration files: 001_add_column.sql, 002_create_table.sql
  • _MIGRATION_HISTORY table in BigQuery to track what’s been applied
  • Runs pending migrations in order
  • Supports --dry-run and rollback
